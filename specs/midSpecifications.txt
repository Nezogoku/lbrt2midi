MIDI HEADER, 0x0E Bytes
    MThd, 0x00
    Size of header (Always 6), 0x04
    Format of file, 0x08
        0 = 1 track chunk
        1 = several vertically dimensional track chunks
        2 = several horizontally dimensional, individual track chunks
    Number of tracks, 0x0A
    Division, 0x0C
        If SMPTE:
            1 fffffff tttttttt
                1 is negative bit
                fffffff is fps
                    24, 25, 29 (or drop 30), 30
                tttttttt is ticks per frame
        If PPQN:
            0 ttttttttttttttt
                ttttttttttttttt is Pulse per quarter-note (PPQN)
                    96 = quarter-note
                    48 = eigth-note
                    24 = sixteenth-note
                    etc...

MIDI TACK HEADER, 0x08 Bytes
    MTrk, 0x00
    Size of track, 0x04

MIDI TRACK, Variable Bytes
    Delta Time
        ~Microseconds = Delta Time * ((Tempo / Division) / 1000)~
        Variable Length Value
        Number of ticks after previous event, Depends on MS bit of byte
            0 = no more bytes in delta time
            1 = at least one more byte in delta time
    Event
        Status byte, 0x01 Byte
            0x8X is Note off (0x8X 0xNN 0xVV)
                X is channel
                0xNN is Note (0x00 to 0x7F)
                0xVV is Release velocity (0x00 to 0x7F)
            0x9X is Note on (0x9X 0xNN 0xVV)
                X is channel
                0xNN is Note (0x00 to 0x7F)
                0xVV is Velocity (0x00 to 0x7F)
            0xAX is Key pressure (0xAX 0xNN 0xPP)
                X is channel
                0xNN is Note (0x00 to 0x7F)
                0xPP is Applied pressure (0x00 to 0x7F)
            0xBX is controller (0xBX 0xCC 0xVV)
                X is channel
                0xCC is Continuous controller
                0xVV is Controller value (0x00 to 0x7F)
                    If Default, Ranged
                    If None, Always 0x00
                    If On/Off, 0x40+ is On and <0x40 is Off
            0xCX is Programme change (0xCX 0xBB)
                X is channel
                0xBB is Selected programme (0x00 to 0x7F)
            0xDX is Channel pressure (0xDX 0xPP)
                X is channel
                0xBB is Applied pressure (0x00 to 0x7F)
            0xEX is Pitch bend (0xEX 0xSSSS)
                X is channel
                0xSSSS is Pitch value (0sssssss 0sssssss)
                    14bit value (0x0000 to 0x7F7F)
                    0x0000 is Usually 2 semitones lower
                    0x4000 is No change
                    0x7F7F is Usually 2 semitones higher
            0xF0 is System exclusive start (0xF0 0xSS 0xII 0x...)
                ~Must end in 0xF7 unless segmented~
                ~Status byte may end segmented SysEx event~
                0xSS is Size of message
                    ~Excluded when transmitted~
                0xII is Manufacturer ID or Realtime message
                0x... is Message
            0xF1 is MIDI time code quarter frame (0xF1 0xQQ)
                0xQQ is Message (0xMV)
                    M is Message type
                        0 = Frame (LS nibble)
                        1 = Frame (MS nibble)
                        2 = Seconds (LS nibble)
                        3 = Seconds (MS nibble)
                        4 = Minutes (LS nibble)
                        5 = Minutes (MS nibble)
                        6 = Hours (LS nibble)
                        7 = Hours and frame rate (MS nibble)
                    V is Message value
                        000fffff is Frame (0 to 29)
                        00ssssss is Seconds (0 to 59)
                        00mmmmmm is Minutes (0 to 59)
                        0sshhhhh is Hours and frame rate
                            ss is Fps
                                00 = 24Fps
                                01 = 25Fps
                                10 = 29.97Fps
                                11 = 30Fps
                            hhhhh is Hours (0 to 23)
            0xF2 is Song position pointer (0xF2 0xPPPP)
                0xPPPP is Position (0ppppppp 0ppppppp)
                    14bit value (0x0000 to 0x7F7F)
            0xF3 is Sequence select (0xF3 0xSS)
                0xSS is Song ID
            0xF4 is Undefined Common System message
                ~Status byte may end canceled segmented SysEx event~
            0xF5 is Undefined Common System message
                ~Status byte may end SysEx event if 0xF7 excluded~
            0xF6 is Tune request
            0xF7 is System exclusive continue (0xF7 0xSS 0xII 0x...)
                ~May end in 0xF7~
                ~Status byte must end final [segmented] SysEx event~
                0xSS is Size of message
                    ~Excluded when transmitted~
                0xII is Manufacturer ID or Realtime message
                0x... is Message
            0xF8 is MIDI clock
                ~Embedded in SysEx event in MIDI files~
            0xF9 is Undefined Realtime message
                ~Embedded in SysEx event in MIDI files~
            0xFA is MIDI start
                ~Embedded in SysEx event in MIDI files~
            0xFB is MIDI continue
                ~Embedded in SysEx event in MIDI files~
            0xFC is MIDI stop
                ~Embedded in SysEx event in MIDI files~
            0xFD is Undefined Realtime message
                ~Embedded in SysEx event in MIDI files~
            0xFE is MIDI connection exists
                ~Embedded in SysEx event in MIDI files~
            0xFF is Reset (0xFF) or Meta event (0xFF 0xTT 0xSS 0x...)
                ~Reset embedded in SysEx event in MIDI files~
                0xTT is Meta type
                    0x00 is Sequence number (0xFF 0x00 0x02 0xIIII)
                        ~Must occur at beginning of track~
                        0xIIII is Sequence number
                    0x01 is Text event (0xFF 0x01 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x02 is Copyright (0xFF 0x02 0xSS 0x...)
                        ~Must occur at beginning of first track~
                        0xSS is Size of string
                        0x... is String
                    0x03 is Track name (0xFF 0x03 0xSS 0x...)
                        ~Must occur at beginning of track~
                        0xSS is Size of string
                        0x... is String
                    0x04 is Instrument name (0xFF 0x04 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x05 is Lyrics (0xFF 0x05 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x06 is Marker (0xFF 0x06 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x07 is Cue (0xFF 0x07 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x08 is Patch name (0xFF 0x08 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x09 is Device name (0xFF 0x09 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x0A is Unknown text event (0xFF 0x0A 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x0B is Unknown text event (0xFF 0x0B 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x0C is Unknown text event (0xFF 0x0C 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x0D is Unknown text event (0xFF 0x0D 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x0E is Unknown text event (0xFF 0x0E 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x0F is Unknown text event (0xFF 0x0F 0xSS 0x...)
                        0xSS is Size of string
                        0x... is String
                    0x20 is Channel prefix (0xFF 0x20 0x01 0xMM)
                        0xMM is MIDI channel
                    0x21 is MIDI port (0xFF 0x21 0x01 0xPP)
                        0xPP is MIDI port
                    0x2F is End of track (0xFF 0x2F 0x00)
                        ~Must occur at end of every track~
                    0x51 is Tempo (0xFF 0x51 0x03 0xTTTTTT)
                        ~Should occur in first track~
                        ~BPM = (TS Denomintor / 4) * (60000000 / Tempo)~
                        0xTTTTTT is Microseconds per quarter-note
                    0x54 is SMPTE offset (0xFF 0x54 0x05 0xHH 0xMM 0xSS 0xRR 0xSR)
                        ~Must occur at beginning of track~
                        0xHH is Hours and frame rate (0sshhhhh)
                            ss is Fps
                                00 = 24Fps
                                01 = 25Fps
                                10 = 29.97Fps
                                11 = 30Fps
                            hhhhh is Hours (0 to 23)
                        0xMM is Minutes (0 to 59)
                        0xSS is Seconds (0 to 59)
                        0xRR is Frames (0 to Fps-1)
                        0xSR is Sub-frames (0 to 99)
                    0x58 is Time signature (0xFF 0x58 0x04 0xNN 0xXX 0xTT 0xQQ)
                        ~Should occur in first track~
                        0xNN is Time signature numerator
                        0xXX is Time signature denominator (Base 2 to the 0xXX power)
                        0xTT is Ticks per click
                            ~Clicks per Note = TPC / PPQN~
                        0xQQ is Amount notated 32nd notes per quarter note (Default is 8)
                    0x59 is Key signature (0xFF 0x59 0x02 0xKK 0xSS)
                        ~Should occur in first track~
                        0xKK is Amount flats(-)/sharps(+) (-7 to 7)
                        0xSS is Scale, 1 byte
                            0 is Major
                            1 is Minor
                    0x7F is Sequencer exclusive (0xFF 0x7F 0xSS 0x...)
                        0xSS is Size of data
                        0x... is Data

    0x03 is Notation Information (0x7F 0x03 0xXX)
        0x01 is Bar number
        0x02 is Time signature (Immediate)
        0x42 is Time signature (Delayed)
    0x04 is Device control (0x7F 0x04 0x0X)
        0x01 is Master volume
        0x02 is Master balance
    0x05 is Real time MTC cueing (0x7F 0x05 0x0X)
        0x00 is Special
        0x01 is Punch-in points
        0x02 is Punch-out points
        0x03 is Reserved
        0x04 is Reserved
        0x05 is Event start point
        0x06 is Event stop point
        0x07 is Event start points with additional info
        0x08 is Event stop Points with additional info
        0x09 is Reserved
        0x0A is Reserved
        0x0B is Cue points
        0x0C is Cue points with additional info
        0x0D is Reserved
        0x0E is Event name in additional info
    0x06 is MIDI machine control commands (0x7F 0x06 0xXX)
        ~bleh~
    0x07 is MIDI machine control responses (0x7F 0x07 0xXX)
        ~bleh~
    0x08 is MIDI tuning standard (0x7F 0x08 0x0X)
        0x02 is Note change
